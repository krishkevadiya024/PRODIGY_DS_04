# PRODIGY_DS_04

## Task 04 - Sentiment Analysis

### Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes towards specific topics.

### Dataset
Twitter Sentiment Dataset

### Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

### Steps Performed
1. Loaded Twitter sentiment dataset
2. Explored dataset structure
3. Visualized sentiment distribution
4. Converted text into TF-IDF features
5. Trained Logistic Regression model
6. Evaluated model performance using accuracy and classification report

### Output
Successfully analyzed sentiment patterns and built a sentiment classification model.
